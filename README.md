## 🎨 KoboldCpp UI Color & Blur Theme

This project provides a customizable color and blur theme for KoboldCpp.  
It enhances the visual appearance of the UI without changing its structure or layout.

## ✨ Features

- 🎨 Customizable UI colors  
- 🌫️ Smooth blur (glass-like) effects  
- ⚙️ No structural changes to the original UI  
- 🔧 Easy to modify and personalize
- 🎨 Dynamic Color Adaptation  
  The UI automatically adjusts its color scheme based on the selected wallpaper, creating a consistent and immersive look.

- 🌫️ Advanced Blur Effects  
  Carefully implemented blur (glassmorphism-style) effects for a smooth and modern appearance.

- 🔘 Smooth Button Animations  
  Deeply customized button animations with responsive and interactive feedback.

- 🧊 3D Visual Style  
  Subtle 3D elements and depth effects to enhance the overall UI feel.

- 🌑 Soft Shadows  
  Soft shadow rendering for better layering and visual clarity.

- ⚡ Highly Dynamic Design  
  The UI reacts to visual elements and feels alive rather than static.

## 🧩 Customization

Users can freely adjust colors, blur intensity, and other visual elements by editing the CSS code.

## 🚀 Installation Guide

To install this UI color and blur theme:

1. Open KoboldCpp  
2. Go to **Settings**  
3. Navigate to the following options:
   - Format  
   - Samplers  
   - Media  
   - Tokens  
   - Advanced  

4. Click on **Advanced**  
5. Find the **Custom CSS** option  
6. Click on **Apply Custom CSS**

You will see two options:

### 🔹 Option 1: Direct Code Method
1. Open the `style.css` file  
2. Copy the CSS code  
3. Paste it into the Custom CSS field  
4. Apply changes  

### 🔹 Option 2: File Upload Method
1. Click on **Install from file**  
2. Select the `style.css` file  
3. Apply the changes  

---

## 📌 Note

This theme only modifies colors and blur effects.  
It does not change the UI structure or core functionality of KoboldCpp.
